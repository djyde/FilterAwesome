FilterAwesome
=============

FilterAwesome is a CSS filter library. With it, you just need to add a class, that you can add a filter to a image.

Try
=======
http://djyde.github.io/FilterAwesome/

Contribute
=======
This is a new project but I will push more beautiful filter later. If you wanna contribute one or more, just:

* Fork this repo
* Create a branch named the name of your filter
* Pull request
